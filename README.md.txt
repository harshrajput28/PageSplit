# PageSplit 📄

A tool built for students to split their practical lab manuals into separate Color and B&W PDF files to save printing costs!

## 🚀 Features
- 100% Private: Files never leave your computer.
- Auto-Detect: Uses pixel math to automatically find color pages.
- Supports PDF and Word (.docx) files.

## 🛠️ Built With
- PDF.js
- pdf-lib
- mammoth.js
- Vanilla JavaScript, HTML, and CSS
