# PageSplit 📄

[cite_start]PageSplit is a 100% private, browser-based tool that helps students separate their practical lab manuals into separate Color and B&W PDF files to save on printing costs! [cite: 9, 18, 41]

## 🚀 Features
- 🔒 **100% Private:** Files never leave your device. [cite_start]All processing happens locally[cite: 41].
- [cite_start]✨ **Auto-Detect:** Automatically scans and finds color pages using pixel analysis[cite: 31].
- [cite_start]📄 **File Support:** Supports both PDF and Word (.docx) files[cite: 25].

## 🛠️ How it was built
- [cite_start]HTML5, CSS3, and Vanilla JavaScript (ES6+) [cite: 22, 59]
- [cite_start]**PDF.js** (for page rendering) [cite: 59]
- [cite_start]**pdf-lib** (for creating the split PDFs) [cite: 59]
- [cite_start]**mammoth.js** (for Word document preview) [cite: 59]
